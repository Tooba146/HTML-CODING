Q1: What does HTML stand for?

Ans: HTML stands for "HyperText Markup Language".

Q2: What is HTML used for?

Ans: HTML HyperText Markup Language is used to create Web pages and tells the browser how to display them. It designs the basic layout and formatting of Web pages.
HTML is made up of elements or tags and attributes which work together to identify document parts and tell the browser how to display them.

Q3: What are HTML elements?

Ans: The HTML element is everything from the start tag to the end tag.
Elements are the "Building blocks" of a web page. An element typically consists of:
1. Opening tag
2. Content
3. Closing tag
